1) What is html for?

HTML is used for web development. Almost all web pages uses HTML as its markup language. HTML means HyperText Markup Language.

HTML is the most popular language used in web development. And the good news is that HTML is relatively easy to learn.

You can think in it as a simple Word document in which instead of clicking on bold or italic buttons. You have to write that as commands inside your text. Take a look below:

2) So, lets see some Basic HTML Tags

HTML tags are what we use to tell a web browser how to display content in a web page. Most HTML tags consist of an opening and a closing tag.

So...

for bold you will use < b > and < / b >

for italic you will use < em > and < / em>

The slash is used to close the command you have opened before.

More HTML tags:

< p >text< / p> Defines a paragraph. The content of the < p > tag is in a "box"

< br > Creates a line break

< div > content < /div> Creates a "box" containter element

< span>content< /span> Creates an "inline" container element

There is a difference between Inline and Block Elements. Inline and Block are HTML Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them, stretching out to the left and right as far as possible. Block elements always start a new line while Inline elements does not. Inline elements only uses as much width as necessary.
